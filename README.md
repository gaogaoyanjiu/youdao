## 开发过程
```
使用vue-cli构建vue项目，添加vue-cli-plugin-electron-builder插件进行开发和打包
```

## 体验项目
```
选择releases中最新版本下载exe文件，暂不支持mac
```

## clone项目

### 开发

```
yarn run electron:serve
```
### 打包

```
yarn run electron:build
```
